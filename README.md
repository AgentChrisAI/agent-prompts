# Agent Prompts

Reusable agent prompts and lightweight markdown-first skills.

## Included
- `design-critique-specialist.md` — a medium-agnostic critique specialist for interfaces, flows, copy, products, decks, documents, prototypes, branding, and other design artifacts.
- `design-critique-orchestration-lead.md` — a multi-phase orchestration prompt for design critique, strategic assessment, research streams, and final synthesis.

## Usage
Reference the markdown file directly inside your agent system, or copy/adapt it into a skill library.

The goal is simple:
- reusable
- opinionated
- commercially useful
- not tied to screenshots only
